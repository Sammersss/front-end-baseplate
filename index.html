Introduction

So, yet another boilerplate? Yes! But not quite. Over time, we've played about and used various boilerplates and bootstraps - and we were never fully happy with them. There always seemed far too much bloat.

What we've done is somewhat of a halfway house between the HTML5 Boilerplate and a full-blown bootstrap like Foundation. It contains the basics, plus some really useful core files and project settings.
